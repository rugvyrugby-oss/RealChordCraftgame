## VoxMorph — voice changer plugin for Ableton Live

The [`ableton-voice-changer/`](ableton-voice-changer/) folder contains
**VoxMorph**, a VST3/AU voice-character transformer (pitch + formant shifting,
drive, doubler, robot/whisper modes) you can build and load in Ableton Live.
See [`ableton-voice-changer/README.md`](ableton-voice-changer/README.md) for
build and usage instructions.
